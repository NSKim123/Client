윈도우 콘솔 응용프로그램의 시작점
-> main 함수 선언

프로그래밍 언어의 기본 구성요소 3가지
	타입 : 메모리를 어떻게 무엇인가로 다룰건지에 대한 규칙
	연산자 : 임으의 두 개의 대상을 임의의 처리를 통해 하나의 결과를 만들어내는 것
	제어구조 : 프로그램의 '실행의 흐름(프로세스)'을 제어하는 구조
		순차처리
		판단
		반복

변수는 선언할 때 태어난다.
함수 : 코드의 추상화. 기능 부품
함수 사용의 3단계
	선언 : 호출call하는 측에 형태를 알려주는 것이다.
	정의 : 실제 함수의 동작을 구현한다.
	호출 : 함수를 실행하는 것이다.

포인터 변수 : 주소를 값으로 가지는 변수.




프로세스 주소 공간 : 코드 - 데이터 - 힙 - 스택

IP(Instrction Pointer) : 명령문의 주소를 담고 있는 포인터 변수

BP(Base Pointer) : 지역의 시작 부분의 주소를 담고 있는 포인터 변수
SP(Stack Pointer) : 지역의 끝 부분의 주소를 담고 있는 포인터 변수

스택프레임 : 이 동작에 의해서 지역변수, 매개변수는 메모리 관리가 자동으로 된다.
